# Literature Review: Brain-Mediated Music and Creativity

This folder contains curated references to key papers organized by research area.

## Organization

- `neural_decoding.md` - Neural decoding and Mind-Vis related papers
- `generative_audio.md` - Generative audio and music models
- `creativity_neuroscience.md` - Neuroscience of creativity and music
- `bci_music.md` - Brain-computer interfaces in musical contexts

## Key Themes

1. **Neural Decoding**: Understanding how to extract meaningful features from brain signals
2. **Generative Models**: State-of-the-art approaches to audio and music generation
3. **Creativity Research**: Psychological and neural basis of creative processes
4. **Applied Systems**: Real-world implementations of brain-music interfaces

## How to Use

Each markdown file contains:
- Paper citations with links (when available)
- Brief summaries of key contributions
- Relevance to this project
- Important findings and methods

## Contributing

When adding new papers, include:
- Full citation (authors, year, title, venue)
- DOI or arxiv link
- 2-3 sentence summary
- Key relevance to project goals
